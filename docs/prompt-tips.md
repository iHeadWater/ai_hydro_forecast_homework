# Prompt 编写技巧

> 好 Prompt 是高效使用 AI Agent 的关键。本章教你怎么写、怎么写得好。
> 本课程用内置 **songliao-calibration** skill，你多数时候只需用自然语言说清"改哪个参数"。

## 核心原则

### 1. 明确说你要什么，不要什么

| ❌ 差的 Prompt | ✅ 好的 Prompt |
|---------------|---------------|
| "帮我做个水文分析" | "帮我率定碧流河流域（songliao_21401550），用 xaj_mz 模型，跑完把 NSE 报给我" |
| "改一下参数" | "把 SCE-UA 的 rep 从 5000 调到 8000，重跑对比碧流河的 NSE" |
| "换个模型" | "把 model_cfgs.name 换成 xaj，在碧流河上重跑率定+评估对比" |

### 2. 给上下文，不要假设 AI 什么都知道

```
"率定配置在 hydromodel/configs/songliao_event_3h.yaml，
数据是 songliao 3h 洪水事件，用 XAJ 系列模型。
从 hydromodel/ 目录运行脚本。"
```

（有 songliao-calibration skill 时，这些 AI 一般已经知道，直接说需求即可。）

### 3. 分步骤，别一口气全说

不要一次说：
> "帮我换流域换模型加大迭代然后画图"

而是拆成多轮对话：
1. "先用默认配置率定碧流河（songliao_21401550），报 NSE"
2. "好，把 rep 调到 8000 重跑对比"
3. "再换成 xaj 模型跑一次"
4. "最后画个降雨-径流过程图"

### 4. 让 AI 先解释再执行

加上这句话可以大幅提高成功率：
> "在改配置/写代码之前，先告诉我你打算怎么改，我确认后再执行"

## 常用术语

在 Prompt 里用这些词，AI 更容易理解你的需求：

| 类别 | 术语 |
|------|------|
| 模型 | XAJ 系列：xaj（新安江）、xaj_mz、semi_xaj、xaj_slw（**3h 事件只用这些，别用 GR/LSTM**） |
| 数据 | songliao 松辽 3h 洪水事件、rain（降雨 mm/3h）、inflow（入库流量 m³/s）、ES（蒸散发）、flood_event（事件标记） |
| 率定 | SCE-UA、rep（迭代次数）、ngs（复合形个数）、kernel_size（汇流单位线长度） |
| 指标 | NSE（纳什效率系数）、KGE、RMSE、PBIAS |
| 阶段 | warmup（预热期，事件数据设 0）、train_period（率定期）、test_period（验证期） |

## 让 AI 自我纠错

善用这些指令让 AI 自己检查：

- "运行这段命令，如果报错就自己诊断修复"
- "你确定这个 NSE 是合理的吗？再检查一遍配置"
- "NSE 大于 1 是不可能的，请检查哪里出了问题"
- "把刚才报的错修好，然后重新运行"

## 常见翻车现场

### 1. 跑错目录

**症状**：报错找不到数据 / 路径不对
**解决**："先 `cd` 到 hydromodel 目录再跑，config 里的 `../data` 是相对该目录的"

### 2. 只看到 RMSE，没有 NSE

**症状**：率定跑完了，但要的 NSE 不见
**解决**："率定只出 RMSE，帮我再跑一步评估（run_xaj_evaluate.py），NSE 在 basins_metrics.csv 里"

### 3. 用了不适配的模型

**症状**：换成 GR4J / LSTM 后跑不通
**解决**："songliao 是 3h 事件数据，只用 XAJ 系列（xaj / xaj_mz / semi_xaj）"

### 4. 改错了不该改的参数

**症状**：改了 `time_unit` / `reader` / `uri` 后报错
**解决**："这几项别动，只改流域、模型、rep、时段这些"

## 进阶技巧

### 让 AI 扮演专家角色

```
"你现在是一个有 10 年经验的水文建模专家。
请帮我分析 songliao 这个流域率定不好的原因..."
```

### 提供参考标准

```
"Moriasi et al. (2007) 的标准是 NSE > 0.5 为满意。
我这个流域率定 NSE 只有 0.3，帮我分析原因和改进方向（调 rep、换 xaj_mz、调 kernel_size）。"
```

### 要求输出格式

```
"结果用表格展示，包含：流域、模型、rep、NSE、KGE。
然后附一段 100 字以内的总结。"
```
